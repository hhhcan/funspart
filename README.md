# funspart

      * 接口数据都来自线上爬取可以直接访问读取的相关公共API.
      
      * 整体采用MVP+RxJava+Retrofit框架.
      
      * 业务方面主要涉及播放、新闻、资讯等三大模块展示.
      
      * UI界面采用Material Design设计规范，tablayout和viewpager配合，viewpager切换fragment，抽屉界面的实现使用DrawerLayout和NavigationView配合.
      
      * 新增全局主题换肤切换响应
